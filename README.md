📊 Social Media Engagement & Campaign Performance Analysis
📌 Project Overview
This project analyzes social media data to understand content performance, audience engagement, and campaign effectiveness across different platforms. Using Python for data preprocessing and Power BI for visualization, raw post-level data was transformed into actionable insights to support data-driven marketing decisions.

🛠️ Tech Stack & Tools

Data Source: Social media dataset (CSV)
Data Cleaning & Processing: Python (Pandas)
Data Modeling: Power BI (DAX)
Visualization: Power BI Desktop

🧠 Key Data Challenges (The "Cleaning" Process)

Working with raw social media data required multiple preprocessing steps:
Handling Missing Values: Cleaned null values in fields like mentions and engagement-related attributes
Data Transformation: Converted timestamps into usable formats (day, hour) for time-based analysis
Feature Engineering: Derived metrics such as engagement rate and posting time patterns
Data Consistency: Standardized categorical fields like platform, sentiment, and campaign phases

📈 Key Insights
Content Performance: Delivery-related content generated the highest engagement
Optimal Timing: Friday around 12 PM emerged as the most effective posting time
Sentiment Impact: Neutral and informative content outperformed highly emotional posts
Campaign Effectiveness: Engagement varied across campaign phases, highlighting optimization opportunities
